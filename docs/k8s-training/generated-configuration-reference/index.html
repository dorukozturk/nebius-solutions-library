KubeRay Cluster​

enable_kuberay_cluster​

for GPU isolation to work with kuberay, gpu_nodes_driverfull_image must be set to false. This is because we enable acess to infiniband via securityContext.privileged Turn KubeRay to false, otherwise gpu capacity will be consumed by KubeRay cluster

Example from terraform.tfvars:

enable_kuberay_cluster = false # Turn KubeRay to false, otherwise gpu capacity will be consumed by KubeRay cluster
